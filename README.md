# python-basics
Python practice repository for students of EarthSystem-Science-Lab, providing hands-on exercises, notebooks, and coding tasks to build foundational programming skills.
Instructions to Students:
Step 1: Accept the GitHub invitation. You will receive a GitHub invite from: EarthSystem-Science-Lab (Organization). Accept it. If you do NOT have a GitHub account, create one first: https://github.com
Step 2: Fork the python-basics repository. Visit https://github.com/EarthSystem-Science-Lab/python-basics . Click the Fork button (top-right corner). This creates a personal copy under your account: e.g. https://github.com/<your-username>/python-basics . This is YOUR workspace. All your practice work will remain inside your fork.
Step 3: Open any notebook in Google Colab. Inside your forked repo: Go to the folder notebooks/ . Click on any notebook 01_Python_Fundamentals.ipynb . Click the Open in Colab button. You will now have the notebook open in Google Colab.
Step 4: Save a copy into your own Drive: File → Save a copy in Drive
Step 5: Complete all exercises in /notebooks/ with displayed outputs. After you finish each exercise successfully in Collab, Go to: File → Download .ipynb . Save it to your computer.  Upload the completed .ipynb to your fork: In your repository (forked) → Upload files (e.g., notebooks/01___.ipynb) and click on Commit changes.
